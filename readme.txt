> HOW TO USE

1. Search your car in "Model" dropdown
2. Select your desired fuel in "Jenis BBM" dropdown
3. Paste value from "model" and "jenis bbm" for "Konsumsi BBM" and "Harga BBM" form
4. Input query for "Asal"
5. See the result


> IMPORT DATABASE & RUN THE PROGRAM IN LOCAL MACHINES

1. Copy "OnCost" project folder to C:\xampp\htdocs\RPL\ so the working folder will be C:\xampp\htdocs\PROGIF\OnCost
2. Turn on XAMPP localhost
3. Create new database in localhost phpmyadmin, and name it as 'oncost'
4. Import SQL dump file from db folder named 'oncost.sql' into oncost database
5. Run the program from your browser in http://localhost/PROGIF/OnCost/
